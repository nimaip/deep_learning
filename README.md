# Deep Learning Architecture Implementations

## Overview
This repository contains practical implementations of various deep learning models and architectures. It serves as a code-first log of translating theoretical machine learning concepts into functional, trainable networks. 

## Scope of Work
The implementations in this repository progress through different families of deep learning models, focusing on understanding the underlying mechanics, optimization bottlenecks, and training dynamics of each architecture. 

Key areas of focus include:
* **Sequence Modeling:** Vanilla RNNs, LSTMs, GRUs, and early attention mechanisms.
* **Transformers:** Encoder/decoder blocks, self-attention flows, and positional encoding.
* **Generative Models:** Autoencoders, Variational Autoencoders (VAEs), and GANs.
* **Computer Vision (CV):** CNN building blocks, ResNet baselines, Object Detectors (YOLO, Faster R-CNN), and Vision Transformers (ViT).
* **Modern Paradigms:** Contrastive learning, self-supervision, and diffusion training loops.

## Tech Stack
* **Language:** Python
* **Primary Framework:** PyTorch
* **Data & Math:** NumPy, standard PyTorch utilities (DataLoaders, transforms)

## Repository Goals
The primary objective of this project is to build a comprehensive, working reference of modern neural networks. Rather than relying solely on high-level APIs, these modules prioritize writing the forward passes, custom loss functions, and core network layers from scratch to solidify architectural understanding.
